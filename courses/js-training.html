專為「程式新手」安排的課程大綱

這門課程，我們是以「 業界實務 JS 技能 」為導向，全程會以 ES6 語法來教學，讓您上完課後，無縫接軌職場。

第一堂：重新認識 JavaScript 之術

學不好 JS 的原因，往往是基本觀念沒紮穩，以致後面越學越混亂，本週將學習最簡單也最重要的變數觀念，儘管你是學習一年以上的開發者，在這堂課上也會受益良多！


教學關鍵字：
變數、型別、陣列與物件、let、const

第二堂：邏輯判斷之術

接下來，我們將開始培養「程式邏輯」，你將寫程式，讓電腦依照你的想法去執行。同時會設計大量的情境題，藉由學習引導更快掌握開發邏輯！


教學關鍵字：
if、else、邏輯運算子

第三堂：物件跟陣列資料處理之術

你有一批資料，想篩選 (filter)、或計算出你想知道的資訊？那你絕對不能不會物件與陣列！本週將會詳細講解資料處理的各種語法，讓你成為資訊處理達人~


教學關鍵字：
陣列、物件、陣列操作大全、for、forEach、JSON

第四堂：函式設計之術

要成為專業的工程師，函式會是你的最大利器，將每個功能都包裝成函式，讓您能有條理地管理程式碼！


教學關鍵字：
function、箭頭函式、參數設計、監聽整合、this

第五堂：期中試煉

不只輸入知識外，你也需要大量輸出練習，讓技術真正地吸收成自己的技能。本週我們將會出各種關卡，讓您能跨越重重邏輯障礙，破繭重生！


教學關鍵字：
JS 開發思維、程式邏輯整合

第六堂：AJAX 操控伺服器之術

AJAX 技術能夠讓您取得與更新遠端伺服器資料，本課程將會提供「專屬 API」，讓您能充分練習！


教學關鍵字：
AJAX、JSON、Axios、fetch、promise、HTTP

第七堂：第三方 JS 套件整合之術

除了自己寫程式碼，站在巨人的肩膀上使用成熟套件加速開發，並有條理地整合也是重要一環。本週將會分享業界實務套件、讓您開發無往不利！


教學關鍵字：
C3.js SVG 圖表、資料驗證、動畫效果

第八堂：RESTful API 整合開發之術

到這階段我們將會學習如何整合串接 API，課程提供 10 支以上的 API，讓您能充份了解 API 介接的各種眉角！


教學關鍵字：
RESTful API、axios

第九堂：期末專案試煉

為了確保您真正學會 JavaScript，我們將會設計「專案級」的期末試煉，並模擬業主需求，讓您深入其境，瞭解前端工程師常見工作情境！


教學關鍵字：
實務開發、JS 中高階觀念

常見 Q & A

Q：這堂課適合新手嗎？

A：適合完全無經驗的新手，會手把手帶您學到會！

Q：JavaScript 工程師養成直播班是預錄的嗎？

A：不是哦，每次課程都會是老師直接 Live 教學，這樣才能提升互動性，增強學員吸收知識體驗

Q：有幾天直播時段沒辦法配合的話，會有錄影影片嗎？

A：有的！我們直播影片都會保留一年時間，方便學員隨時回平台回顧。

Q：我看到你們有影音課程（ex: JavaScript 前端修練全攻略）跟 JavaScript 工程師養成直播班，到底該選哪個好？

A：如果你有迫切性想學好 JavaScript 及一個框架，建議你選本頁面的直播課程，直播的參與互動性高，並藉由老師們獨創的前端教材，學習效果更好。

Q：那影音課程（ex: JavaScript 前端修練全攻略）的話呢？

A：在影音課程（ex: JavaScript 前端修練全攻略）設計上，我們一小節都設計成 5~20 分鐘，方便時間較碎片，沒辦法抽出一小時以上觀看直播課程的學員來選擇。

Q：課程提供的 API 是新的嗎？

A：本次課程 API 是重新開發的，除了架構及功能不同以外，也可以在做作業的過程中發出請求讓我們隨時新增功能喔。

Q：但是我有你們的影音課程（ex: JavaScript 前端修練全攻略），還需要參加 JavaScript 工程師養成直播班嗎？

A：影音課程的設計上，理論和實作都是 5:5，而 JavaScript 工程師養成直播班則是實作機會比較多，互動性也高。如果你是迫切想掌握好一門技術的學員。建議是參與 JavaScript 工程師養成直播班，並藉由影音課程來提升知識量。

Q：除了老師直播外，我有問題該怎麼辦？

A：我們遊戲系統會提供問答專區，線上的助教都會在發問後 48 小時工作日內回覆，並且也會提供專屬的 Discord 聊天頻道，來讓您與助教一起互動！

Q：直播班的作業可以補交嗎？

A：當然可以，只要在 JavaScript 工程師養成直播班開班後的三個月內都可以補交哦～

Q：如果無法參與，可以申請退費嗎？

A：退費前請先詳閱退費規定。若不符合需求仍欲退款，可寄信至 service@hexschool.com，將會有小幫手協助進行退款。
