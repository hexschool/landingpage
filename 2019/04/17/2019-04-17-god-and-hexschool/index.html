【大神來六角】五月直播主題：後端開發

  • 2019/04/17
  • event

同學您好，六角學院固定會在前一個月，釋出下個月的直播節目表到 Google 日曆，歡迎訂閱我們的行事曆，好方便追蹤您想聽的題目。

本月主題：後端開發

後端語言五花八門，各語言都有它的優缺點，但要你真的去選擇一門來研究，是否常常無法決定？所以本月的主題，我們邀請到四位在各後端領域都有一定水準的工程師，來為各位進行分享。

考量到大家沒有後端基礎，六角在與各講師討論後，也決定在主題設計上，也有適合只會 HTML、CSS 網頁排版的初學者也能吸收的內容，以方便讓您能從中選擇適合自己的後端語言。

在演講內容比重中，都會請每位講師分享「什麼是後端，與前端是如何溝通？」，好讓初學者了解後端與前端的關係，進而依照自己所理解的脈絡來延伸學習，藉由每位講師大神的視角來詮釋，能更加瞭解前後端的溝通關係。

在這門主題中，您將會獲得的知識為：

  • 什麼是後端，與前端是如何溝通？ (適合新手，將會在演講比例佔 30~50% 時間)
  • 各後端語言的優缺點，四週講師主題分別為
    • Node.js
    • PHP
    • .Net
    • Ruby

講師介紹

第一週 - Node.js

PJChender 是一位熱愛分享 JS 知識的開發者，目前也是 ALPHA Camp 的學期三 Node.js 講師，許多 JS 開發者都曾從他的部落格獲得 JS 知識，六角學院在兩年前也邀請他分享前端知識，在當時也是大受好評，藉由本月題目，很高興能夠再次邀請他來分享 Node.js 後端的主題！

  • 講座時間：2019/5/7 (二) 19:30~21:00
  • 講者姓名：PJChender (粉絲團)
  • 講者介紹：PJChender 是一個關注網頁前端設計、技術，學習 know how 的社群分享平台，並且以開發者友善的概念為核心，提供開發者需要的訊息和內容。
