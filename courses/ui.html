UI 設計與平面設計的大差異

平面設計

內容無法被操作互動

介面設計

內容可以被互動操作

平面設計的成品會在輸出物上，當輸出後畫面則不再更動；UI 設計的成品則會出現在電腦及行動裝置裝置上，且會隨著用戶操作不斷的改變，因此如何引導用戶正確操作是介面設計師的重要課題。

常見 Q&A

Q: 這堂課需要有設計基礎才能上嗎？

A: 這堂課本身就是為了 UI 領域的初學者所設計，所以並不需要有設計背景。只有在實作的部分會需要使用介面繪圖軟體，如 Adobe XD。

Q: 這堂課會教到 UX（使用者體驗）嗎？

A: UI 與 UX 息息相關，課程裡面會帶到一些設計準則，能夠幫助優化介面去提升使用者體驗，但並不會直接教授 UX 的研究方法等等。

Q: 這堂課是以 Web 還是 APP 的 UI 為主呢？

A: 課程的實際演練是以 Web UI 為主，而課程裡面會介紹 UI 領域通用的知識，適合想要了解 UI 的初學者。
