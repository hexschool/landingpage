常見 QA

  • Ｑ 今年只有這一班嗎？

    目前只有規劃這一班，今年將全心投入在有強烈動機轉職的開發者。

  • Ｑ 這堂課是線上課還是線下？

    為純線上直播課程。

  • Ｑ 我目前在職和在學，適合參加嗎？

    適合參加，原因有二：
    一、直播時間為週五晚上 20:00，並會提供直播錄影
    二、所有教學資源在開課當天，就會全部釋出，你不用等到我們上課時間，自己就能開始進行預習

  • Ｑ 你們會如何專題團隊分組？

    報名後我們會提供能力分組表單，會將年資相近的開發者盡可能分組在一起，協助小組內溝通順暢。

  • Ｑ 有分期付款機制嗎？

    有的，藍新金流提供 6 期 0 利率的分期付款，支援銀行有玉山銀行、台新商銀、中國信託銀行。除此之外還有提供銀角無卡分期，讓您不需要任何信用卡也能享有分期服務。

  • Ｑ 請問開通後相關影片有觀看期限嗎？

    預習影音與直播影音的觀看期限一致，
    切版直播學程、切版專題學程期限至 2025/06
    JavaScript 學程期限至 2025/09
    React 學程、框架專題學程期限至 2025/12

  • Ｑ 本課程是預錄的嗎？

    不是哦，除了課前的預習影音之外，老師也會 Live 教學，這樣才能提升互動性，增強學員吸收知識體驗

  • Ｑ 有幾天直播時段沒辦法配合的話，會有錄影影片嗎？

    有的！我們會在課後的隔天上傳影片，讓同學可以隨時回顧課程的內容

  • Ｑ 除了老師直播外，我有問題該怎麼辦？

    六角教學系統會提供問答專區，線上的助教都會在發問後 48 小時工作日內回覆，並且也會提供專屬的 Discord 聊天頻道，來讓您與助教一起互動！

  • Ｑ 請問學習卡使用方法和功能？

    註冊 “切版直播班” 相關課程，可以獲得實體學習小卡，該卡片是採用刮刮卡的形式，當你完成特定課程挑戰時，即可將該刮刮卡刮開，並統計該分數，增加個人學習成就感。

  • Ｑ 請問我可以使用平板電腦或是智慧型手機上課嗎？

    我們提供影片及直播的形式上課，直播的內容則是使用 Zoom 服務，因此在手機上也能正常運行。
    但在此也額外提醒，由於課程包含許多程式碼，使用行動裝置可能會顯得程式碼較小的狀況哦。

  • Ｑ 請問完課證明領取資格？

    每週的直播課後都會提供作業，在課程結束之前提交所有作業並交由助教審核成功，即可拿到完課程證明喔。
