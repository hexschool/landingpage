直播教室：使用 ZOOM 進行直播教學

4/8
第一週：MongoDB 資料庫管理

要成為後端工程師，資料庫設計是絕對必修項目！MongoDB 是一種文件導向的 NoSQL 資料庫管理系統，除了教您基本的 CURD 增刪查改，更會教您依專案特性來規劃資料表！

本週作業：動態牆資料表設計

MongoDB Compass GUI Altas 雲端資料庫

週一至週四每日任務

4/15
第二週：Node.js NPM 整合 MongoDB

為了讓後端能整合資料庫，我們將會使用 Mongoose NPM 套件，教您整合至 Node.js 應用程式！

本週作業：最終關卡資料表設計

Node.js NPM Mongoose

週一至週四每日任務

4/22
第三週：從 express MVC 架構建立 RESTful API

express 是 Node.js 最成熟的 web 框架，讓我們站在巨人的肩膀上，一窺開發 CSR (Client Side Render) 的 SPA 單頁式網站應用網站架構！

本週作業：開發 最終關卡 GET API

Expres router MVC SPA

週一至週四每日任務

4/29
第四週：期中考｜打造全端 (Full Stack) 網站架構

我們將提供專屬 UI 設計稿，並將採用 Heroku 雲端主機，同時搭配您設計的 MongoDB RESTful API，手把手帶您做出前後端整合的全端架構！

本週作業：期中考｜前後端分離開發

Heroku Postman

週一至週四每日任務

5/6
第五週：Express middleware 異常狀態處理

凡事沒有一帆風順，客戶永遠會用各種神操作，去玩弄你的網站。這堂課我們將鉅細彌遺介紹， Node.js、MongoDB 常見異常狀態管理，並適時回傳 HTTP 狀態碼 200、400、500 錯誤到 Client 進行處理~

本週作業：Swagger API 文件設計

middleware validation HTTP 狀態碼

週一至週四每日任務

5/13
第六週：JWT 身份驗證機制

為我們的網站添加註冊、登入功能吧！這堂課將用業界最熱門的 JWT 來進行身份驗證。同時也會教您會員頁面的權限管理設計！

本週作業：實作一個登入註冊介面吧！

JWT Token 金鑰設計

週一至週四每日任務

5/20
第七週：Imgur 第三方圖床服務 API 介接

上傳圖片是一個網站必要功能！我們將介接 Imgur 第三方服務，讓您的網站增添更多使用者互動！並且傳授前後端介接第三方 API 的獨門重要心法！

本週作業：整合登入驗證，實作圖片上傳功能

API AJAX 檔案上傳 dotenv 切換測試/正式變數

週一至週四每日任務

5/27
第八週：期末考：實作一個 FB、IG 牆吧！

我們將提供多達 10 頁以上的專屬 UI 設計稿，手把手挑戰能登入註冊，上傳心情圖片，還能在別人貼文上按讚留言的的互動牆，為自己的特訓留下完美 ending！

本週作業：實作一個 FB、IG 牆吧！

如果以下都沒有解決到您的問題...

立即聯絡我們的 線上客服 ，專人協助解決您的疑問！

Q：要上這堂直播班，JavaScript 需要會到什麼程度？

A：至少得會用 AJAX 去取得 API 資料，可以花三分鐘做此測驗問卷，分數在 60 分以上就達授課門檻

Q：我做了測驗，發現這班報名是需要資格審核制？

A：是的，本班目前已招生 160 位，絕大部分都是有 1~2 年前端工作經驗的開發者，為確保團體授課品質，本門課將採資格審核制。

Q：我填寫問卷了，什麼時候會獲得符合資格通知？

A：當您註冊問卷後的工作日 48 hr 內，就會收到我們的 Email 通知。

Q：當我收到通知後，何時需註冊課程？

A：付款時間為 3/15(二) 21:00~3/16(三) 23:59:59，將會於 3/15(二) 21:00 發出 Email 通知，提供專屬付款連結申請報名， 若逾期則不保留名額

Q：Node.js 直播班是預錄的嗎？

A：不是哦，每次課程都會是老師直接 Live 教學，這樣才能提升互動性，增強學員吸收知識體驗。

Q：有幾天直播時段沒辦法配合的話，會有錄影影片嗎？

A：有的！我們直播影片都會保留到 2022/9/27，方便學員隨時回平台回顧。

Q：除了老師直播外，我有問題該怎麼辦？

A：我們遊戲系統會提供問答專區，線上的助教都會在發問後 24 小時內回覆，並且也會提供專屬的 discord 聊天頻道，來讓您與助教一起互動！

Q：如果無法參與，可以申請退費嗎？

A： 退費前請先詳閱 退費規定。若不符合需求仍欲退款，可寄信至 service@hexschool.com，將會有小幫手協助進行退款。
