使用 HTML、CSS 開發一個網站

主打 HTML、CSS 課程，上線不到四個月，學員總數破 1500 的線上影音課程。

搶先預覽課程

講師：廖 洧杰

  • 高雄前端社群發起人
  • 中華電信前端講師
  • 高雄大學前端講師
  • 十年實務經驗，授課時數超過一千小時

我將從這堂課程學到什麼？

  1. 有能力排出具專業、語意化、美觀的網頁設計
  2. 課程結束後，我們會提供給你一個完整的網頁設計稿，當你實作出網頁來後再轉交六角學院的講師們，我們會為您進行程式碼審視，以確保你吸收的網頁排版觀念是正確的。
  3. 了解設計網頁的脈絡，你隨時可以在社群裡發問問題，我們永遠都會在這裡！
  4. 獲得持續更新的講義教材，終身複習不會脫離趨勢的現代化線上課程

  • 本課程適用於 Windows、OSX、Linux 的電腦系統
