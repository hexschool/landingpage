常見 Q&A

Q：這堂課程要到什麼水準才適合上呢？

A：至少掌握 JavaScript 基礎知識，如變數、函式、型別等觀念，或是參與 「JavaScript 前端修練全攻略」。

Q：我有買 Vue 出一個電商(v2 版本)，與 Vue 3 課程上有什麼差異呢？

A：兩者課程都是著重在該版本的介紹，兩個課程的架構上差異不大，Vue 3 課程算是基於過去的優化，新增內容包含：

  • Vue 3 專屬的特定語法，如：Composition API、Teleport、新的套件引用。
  • 新的課程架構，將 JS 必備的核心觀念移至課程最前方，如 this、物件傳參考、關注點分離等知識。
  • 基於過去的版本，新增更多的 API 可供串接，總 API 數量達 30 隻（包含前後台、產品模組、訂單模組、驗證模組、貼文模組）。

Q：單日直播與課程內容相同嗎？

A：是的，單日直播目的是讓大家可以快速開始 Vue3.js，會有獨立的課程大綱用不同觀點進行切入，也會加入部分單日直播專屬的主題喔～

  1. 引導：必學的 JS 觀念
  2. 引導：快速掌握 Vue3.js
  3. 直播專屬主題：作品實戰技巧（設計）
  4. 直播專屬主題：Vue3 Proxy 的運作原理

Q：Vue 出一個電商(v2 版本) 會額外新增 Vue 3 的介紹嗎？

A：會的，我們會在未來幾個月上線 Vue 2 轉換 Vue 3 技巧於課程之中。

Q：一定要 Vue2，才會 Vue3 嗎？

A：不用，可以直接學 Vue3，其知識量有些是相通。

Q：我剛學 JavaScript，那我現在該學 Vue2 還是 Vue3？

A：

  1. 如果你還沒有接觸過框架，會建議直接學習 Vue3。
  2. 如果你已經學一半我們的「Vue 出一個電商影音課程」，建議你繼續學。原因是目前 Vue2 還是佔大宗。再者，學完 Vue2 再學 Vue3 掌握度更快，兩者也蠻相像，只是 Vue3 寫法變得更靈活更強大。

Q：這門課程學完，就能獲得獨立的作品嗎？

A：可以的，只要按照課程的流程完成每個章節作業，最終就能完成屬於自己的獨立作品。

Q：作品會是以怎樣的形式呈現呢？

A：每個作品都是獨立的網站，可以放在 Github Pages 的免費空間，更能凸顯個人的專業性。參考 學長姊們的作品。

Q：課程的 API 可以用來作為個人作品來求職嗎？

A：當然可以

Q：如果課程有問題，該如何尋求協助呢？

A：每個章節下方皆有發問區，任何問題都會有講師、助教來回覆，你也可能從其他同學找到你的問題答案喔～

Q：如何參與求職輔導服務？

A：依照課程流程完成最終作業，我們將給予求職輔導表單，詳情可參考「就業輔導如何進行」

Q：此課程未來會持續更新嗎？

A：Vue3 課程已於 2021 年 6 月全數上架課程完畢。課程擴充更新部分，將會在未來三年，因應時代趨勢，每年新增 60 分鐘教材，總計增加 180 分鐘直播錄影或影音課程，並於 2024/12/31 前完成全數更新。
