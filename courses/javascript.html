搶先預覽課程

Web 前端三巨頭 HTML、CSS、JavaScript 各有自己的重要性，其中 JavaScript 更是具有豐富動態功能的語言，除了製作動態效果外，在串接資料更是不可或缺的關鍵，現在職場上前端職缺更是把 JavaScript 列為重點必備項目。

網頁設計互動的要素

JavaScript 是 HTML 三巨頭之中的互動要角，無論是表單驗證、資料傳遞、圖表呈現等。都不可能缺少 JavaScript。

前端框架的邏輯基礎

現在前端開發大多都會使用 JavaScript 框架，如 Angular.js、React.js、Vue.js，如具有 JavaScript 入門基礎，在操作這些框架更能得心應手。

自己動手做延伸更多的不可能

現在 JavaScript 不僅是瀏覽器上執行的語言，更能運行在 Node.js、Arduino、Mobile App 等平台上，達到學一個語言多方的運用！

夥伴一：伴隨學生成長的守護神

進入本課程的學生，六角學院都會提供一個守護神跟你一塊練習，當你越了解 JS 時，守護神也會跟著進化，使得學習更充滿期待與樂趣。

夥伴二：動畫解釋程式結構原理

課程中的觀念都會先用動畫情境導入，理解玩概念後再進入主要課程內容，無論是物件、陣列、迴圈，這些觀念都讓你輕鬆上手。

夥伴三：全新開發，線上編碼練習

學習程式過程中總是會擔心自己的觀念沒搞懂，這次開發了線上編碼練習，讓學生在學習的過程中就能夠驗證自己的觀念，讓線上課程如同實體課程的效益。
