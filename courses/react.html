Q：這堂課程要到什麼水準才適合上呢？

A：至少掌握 JavaScript 基礎知識，如變數、函式、型別等觀念，或是參與「JavaScript 前端修練全攻略」。

Q：React 與 Vue 該選哪一個學習呢？

A：兩者均是目前主流的框架，業界大部分都會從主流框架中擇一選擇運用，因此選擇其中任一均可。

Q：如果課程有問題，該如何尋求協助呢？

A：每個章節下方皆有發問區，任何問題都會有講師、助教來回覆，你也可能從其他同學找到你的問題答案喔～

Q：作品會是以怎樣的形式呈現呢？（作品章節於 2023 年 2 月上線）

A：每個作品都是獨立的網站，可以放在 Github Pages 的免費空間，更能凸顯個人的專業性。在 Vue 3 實戰影音課程中已有大量的範例，可參考學長姊們的作品。

Q：如何參與求職輔導服務？

A：依照課程流程完成最終作業，我們將給予求職輔導表單，詳情可參考「就業輔導如何進行」。

Q：此課程未來會持續更新嗎？

A：此課程預計於 2023 年 3 月課程上架完畢，將會在未來兩年，因應時代趨勢，每年新增 60 分鐘教材，總計增加 120 分鐘直播錄影或影音課程，並於 2025/12/31 前完成全數更新。
