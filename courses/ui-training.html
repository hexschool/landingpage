在學習 UI 時，
你是否有過這些誤解

用越多樣式才是有設計感？

你是否設定了一大推的字級跟樣式，整個網頁看起來都不像是同一個人做的嗎？等到要修改的時候，也常常忘記改到一些地方嗎？

排版全憑感覺就好？

被別人問到設計稿為什麼要這樣排版時，總是只會回答「這樣感覺比較好看」，卻講不出一個更有說服力的具體原因嗎？

以為網頁只需要畫一種尺寸？

你以為畫完一個尺寸之後，其他響應式的尺寸都會自動完成了嗎？還是你甚至從來沒有聽過什麼是響應式網頁設計？

你可能只是少了一些設計觀念

缺乏設計準則

設計準則能夠幫助你讓設計稿有一致性，像是重複使用類似的樣式更能夠建立整體的形象，設計的品味反而體現於這些細節裡。

你需要懂一些程式邏輯

設計師要掌握介面上每個間距與對齊方式，繪製時套用一些程式邏輯，就能夠讓工程師更快理解交付給他的設計稿要如何開發。

響應式的規劃大有學問

響應式是為了讓不同螢幕尺寸下都能有最好的介面呈現，電腦和手機螢幕的長寬差異會大大影響使用者在操作時的習慣和體驗。

常見 Q & A

UI 設計必修觀念直播班是預錄的嗎？

不是哦，每次課程都會是老師直接 Live 教學，這樣才能提升互動性，增強學員吸收知識體驗。

有幾天直播時段沒辦法配合的話，會有錄影影片嗎？

有的！我們直播影片都會保留一年時間，方便學員隨時回平台回顧。

影音課程（e.g. UI 設計入門）跟直播班要選哪個才好？

若是想在較短的時間內建立 UI 設計的基礎觀念，會建議選擇直播班。

但是我有你們的影音課程，還需要參加直播班嗎？

影音課程的設計更著重在理論與知識的完整性，直播班則多了大量實作與即時互動，且針對常見的諸多錯誤觀念進行講解與釐清。如果你是迫切想掌握好一門技術的學員，建議可以參與直播班，並藉由影音課程來提升知識量。

我沒有任何基礎也可以直接上 UI 設計觀念必修直播班嗎？

當然可以，UI 設計觀念必修直播班的課程規劃很適合沒有任何基礎的新手學習，只要有跟上每週的進度，就能順利完成包含最終作品的所有作業。

要上直播班之前我需要先準備什麼？

報名之後會提供課前文件，上面會將你所有需要先下載的軟體等（e.g. Figma）都表列好，只要按照步驟操作就能完成課前準備，且放心，這邊不會有需要額外付費的內容。

這個直播班會有作品嗎？

有的！除了每週的作業之外，還會引導學員從頭製作一個單頁的響應式網站（Landing Page），完成專屬的設計作品。

直播班的作業可以補交嗎？

當然可以，只要在 UI 設計觀念必修直播班開班後的兩個月內都可以補交哦～
